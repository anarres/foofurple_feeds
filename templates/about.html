About the Foofurple Feed Reader

It looks like a website, but it isn't on the Internet; it only exists on your computer

Foofurple Feed Reader is actually a desktop application - a program that runs on your computer, not on an Internet server. However unlike most desktop applications, Foofurple Feed Reader doesn't have a graphical user interface (GUI) of its own. Instead, when you launch Foofurple Feed Reader, it generates some HTML files and displays them in your web browser. We set it up this way because we found that most people would rather use a web browser rather than a stand-alone desktop application. This website that your are reading right now isn't on the Internet; it only exists on your computer, but nevertheless it looks and behaves like a regular website.

How do I update the feeds?

Re-start Foofurple Feed Reader.

Why do I have to save a text file in order to save my settings?

Because Foofurple Feed Reader runs in your web browser, it can't just quietly save files to your computer without telling you first, it has to ask you explicitly to save them.
